Full Practice Exam | Learn Cloud Computing | Pass the AWS Certified Solutions Architect Associate Certification SAA-C02!

## Installing of awscli
AWS is required for this practice for macbook macos only.

```bash
brew install awscli
```

## Do not store ssh keys
It is not safe to store ssh keys even though this is for practice, instead write this command.

```bash
ssh-keygen -f /tmp/ec2
```
